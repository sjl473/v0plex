# 更新日志

本页面记录 v0plex 的版本更新历史。

## 版本说明

v0plex 使用语义化版本号。版本号格式为 `主版本号.次版本号.修订号`。主版本号变更表示不兼容的 API 修改。次版本号变更表示向下兼容的功能新增。修订号变更表示向下兼容的问题修复。

## 历史版本

### 0.1.0

初始发布版本，包含核心功能。

VMD 解析器实现 Markdown 到 React 组件的编译。支持标准 Markdown 语法。支持数学公式（行内和块级）。支持提示框（info、warning、success）。支持文章布局（左右分栏）。支持缩略图。

多语言支持支持多语言内容组织。支持语言切换导航。支持本地化字符串配置。支持语言特定日期格式。

构建系统支持本地和远程内容源。支持 git 历史信息提取（@git 占位符）。支持图片处理和哈希。支持代码块提取。支持语法验证和错误报告。

前端组件响应式布局。侧边栏导航。页面目录。代码语法高亮。MathJax 数学公式渲染。深色主题支持。

## 贡献变更

如果你实现了新功能或修复了问题，请在 Pull Request 中描述变更内容。发布时会将重要变更记录在此日志中。
